# Çift Yönlü Bağlı Liste ile Öğrenci Yönetim Sistemi (C)

## Kullanılan Veri Yapıları

- Doubly Linked List (Çift Yönlü Bağlı Liste)
- Pointer Yapıları
- Dynamic Memory Allocation (`malloc`)
- Array Yapısı
- Quick Sort Algoritması

---

## Projede Yapılan İşlemler

### 1. Öğrenci Ekleme
Yeni öğrenci kayıtları bağlı listeye eklenir.

### 2. Öğrencileri Listeleme
Tüm öğrenciler ekrana yazdırılır.

### 3. Ders Filtreleme
Sadece "Veri Yapıları ve Algoritmalar" dersini alan öğrenciler listelenir.

### 4. Ada Göre Sıralama
Öğrenciler A-Z şeklinde isimlerine göre sıralanır.

### 5. Quick Sort ile Numaraya Göre Sıralama
"Veri Yapıları ve Algoritmalar" dersini alan öğrenciler numaralarına göre sıralanır.

---

## Kullanılan Fonksiyonlar

| Fonksiyon               | Açıklama                                |
|-------------------------|-----------------------------------------|
| `ogrenciEkle()`         | Listeye öğrenci ekler                   |
| `listele()`             | Tüm öğrencileri listeler                |
| `veriYapilariListele()` | Belirli dersi alan öğrencileri gösterir |
| `adaGoreSirala()`       | İsme göre sıralama yapar                |
| `quickSort()`           | Quick Sort algoritması                  |
| `quickSortListele()`    | Filtreleme + sıralama yapar             |
